# 🏠 Real Estate Agent Personal Assistant (REAPA)

## Персональный AI-ассистент для агента по недвижимости

> Универсальный продукт, решающий 50+ выявленных болей агентов. Адаптирован для англоязычного рынка с фокусом на Мальту и международные рынки.

---

## Миссия продукта

**Освободить агента от 60–70% рутины**, чтобы он фокусировался на том, что приносит деньги: переговоры, показы и закрытие сделок.

**Ценностное предложение (USP):**
> «Ваш AI-партнёр, который работает 24/7: находит клиентов, ведёт переписку, готовит документы и анализирует рынок — пока вы закрываете сделки.»

## Модуль 1: 🔍 Lead Engine — Поиск и квалификация клиентов

**Решает проблемы:** лидогенерация, низкое качество лидов, зависимость от порталов, слабый личный бренд

### Функции

| Функция | Описание | Боль, которую решает |
|---------|----------|---------------------|
| **AI Lead Scoring** | Автоматическая оценка лида (hot/warm/cold) по поведению, бюджету, timeline, национальности | Низкое качество лидов, нет приоритизации |
| **Multi-channel Inbox** | Единая лента входящих из WhatsApp, email, портальных форм, Instagram DM, Facebook Messenger | Разрозненные каналы, потеря лидов |
| **Auto-Response Bot** | Мгновенный ответ на запрос в нерабочие часы с квалификацией и записью на показ | Клиенты уходят к конкуренту пока ждут ответа |
| **Referral Tracker** | Трекинг рекомендаций: кто привёл, конверсия, автоматическое thank-you | Нестабильность referral-канала |
| **Lead Nurturing Sequences** | Автоматические цепочки follow-up: email, WhatsApp, SMS — по поведению лида | Клиенты "остывают", пропадают |

### Сценарий использования
```
Лид заполнил форму на портале → REAPA мгновенно отвечает →
квалифицирует (бюджет €300k, ищет 2BR в St Julian's, релокация через 2 мес.) →
присваивает Hot Score → уведомляет агента с рекомендацией: 
"3 подходящих объекта, предлагаю показ в среду"
```

## Модуль 2: 🤝 Client Manager — Управление клиентами

**Решает проблемы:** недоверие, долгий цикл сделки, параллельная работа с конкурентами, невыходящие на связь клиенты

### Функции

| Функция | Описание | Боль |
|---------|----------|------|
| **Smart CRM** | AI-обогащённые карточки клиентов: история взаимодействий, предпочтения, ключевые даты | Данные в блокнотах и WhatsApp |
| **Client Timeline** | Визуальная карта пути клиента: первый контакт → квалификация → показы → оффер → сделка | Нет понимания, где клиент в воронке |
| **Sentiment Tracker** | AI анализирует тон переписки и звонков: клиент доволен, сомневается, раздражён | Эмоциональность решений, срывы сделок |
| **Ghost Client Detector** | Автоматическое определение «пропавших» клиентов + рекомендации по реактивации | Клиенты пропадают без предупреждения |
| **Competitor Alert** | Признаки, что клиент работает с другим агентом (упоминания, снижение активности) | Параллельная работа с конкурентами |
| **Multilingual Chat** | Автоматический перевод переписки EN↔MT↔IT↔RU и другие языки | Языковые барьеры на Мальте |

### Сценарий
```
Клиент не отвечает 5 дней → REAPA: "Maria hasn't responded since showing on Mon. 
Sentiment was 'hesitant' on last call. Recommend: send market update for St Julian's area 
showing 2 new listings below her budget. Draft ready — approve?"
```

## Модуль 3: 🏠 Listing Manager — Управление объектами

**Решает проблемы:** актуальность базы, дублирование, плохой контент, мультиканальное размещение

### Функции

| Функция | Описание | Боль |
|---------|----------|------|
| **One-Click Multi-Publish** | Размещение объекта на 10+ площадок одновременно из одного интерфейса | Ручное размещение на каждый портал |
| **AI Listing Generator** | Генерация описаний на 3+ языках из фото и базовых параметров | Плохие описания, мультиязычность |
| **Photo Enhancer** | AI-улучшение фото, виртуальный staging, удаление мусора с фото | Плохие фото = мало просмотров |
| **Freshness Monitor** | Автоматическая проверка актуальности: объект продан? Цена изменилась? Статус? | Устаревшие объявления |
| **Duplicate Detector** | Выявление дублей своих и чужих объектов на рынке | Дублирование вводит клиентов в заблуждение |
| **Virtual Tour Builder** | Создание виртуальных туров из серии фото (для экспатов, которые не на Мальте) | Экспаты выбирают до приезда |

### Сценарий
```
Агент фотографирует квартиру → загружает в REAPA → AI генерирует:
- Описание на EN, MT, IT
- Улучшенные фото + виртуальный тур
- Публикация на Frank Salt, rightmove.co.uk, idealista, Facebook Marketplace
- Через 30 дней: "Listing viewed 450 times, 12 inquiries. Recommend price adjustment?"
```

## Модуль 4: 📄 Legal & Compliance — Юридика и комплаенс

**Решает проблемы:** AML/KYC, GDPR, регистрация контрактов, EPC, лицензирование

### Функции

| Функция | Описание | Боль |
|---------|----------|------|
| **AML/KYC Checker** | Автоматическая проверка клиента по санкционным спискам и PEP базам | Обязательная due diligence |
| **Contract Template Engine** | Генерация договоров аренды/продажи по актуальным шаблонам Мальты | Ручная подготовка документов |
| **Rental Registration Reminder** | Автоматическое напоминание: зарегистрировать контракт в Housing Authority (30-дневный срок) | Штрафы за просрочку |
| **EPC Tracker** | Контроль наличия Energy Performance Certificate для каждого объекта | Юридический риск |
| **Regulation Updates** | AI-мониторинг изменений в законодательстве Мальты + краткие briefing'и | Частая смена законов |
| **License Renewal Alert** | Напоминания о продлении Property Consultant License, CPD часах | Работа без лицензии = штраф |

### Сценарий
```
Новый клиент-покупатель (non-EU) → REAPA автоматически:
1. Запускает AML-проверку
2. Напоминает об AIP permit (€233)
3. Генерирует чеклист документов
4. Отслеживает сроки подачи
→ Агент видит: "✅ AML clear, ⚠️ AIP pending — remind client in 3 days"
```

## 🎯 Go-to-Market стратегия (Мальта)

### Фаза 1: MVP (Месяцы 1–3)
- **Ядро:** Lead Engine + Client Manager + Task Autopilot
- **Канал:** Direct outreach к 50 крупнейшим агентствам Мальты
- **Hook:** "Бесплатный 30-дневный trial + мы мигрируем ваши контакты из Excel"
- **Цель:** 50 активных агентов

### Фаза 2: Product-Market Fit (Месяцы 4–6)
- **Добавить:** Listing Manager + Content Studio
- **Канал:** Malta Property Forum, MREA events, LinkedIn targeting
- **Hook:** "Автоматическая публикация на 10 порталов в 1 клик"
- **Цель:** 200 агентов, первые платящие клиенты

### Фаза 3: Scale (Месяцы 7–12)
- **Добавить:** Market Intel + Legal/Compliance + Analytics
- **Канал:** Word-of-mouth + партнёрства с Housing Authority, банками
- **Hook:** "Единственный AI-ассистент с Malta-native compliance"
- **Цель:** 500+ агентов, ~30% рынка Мальты

### Фаза 4: International (Год 2)
- **Экспансия:** Кипр, Дубай, Португалия, Испания (похожие рынки: малые, экспат-driven)
- **Адаптация:** Local compliance модули для каждого рынка
