# ЧАСТЬ 3: MVP СПЕЦИФИКАЦИЯ (Фаза 1, месяцы 1–3)

## Scope MVP

**Принцип:** Минимум функций, максимум ценности. Один "wow-момент" на каждый модуль.

### MVP включает 3 модуля:

| Модуль | "Wow-момент" | Метрика успеха |
|--------|-------------|----------------|
| **Lead Engine (lite)** | AI Lead Scoring + Auto-Response в WhatsApp | Время ответа на лид <2 мин |
| **Client Manager (lite)** | Smart CRM + Ghost Client Detector | -30% потерянных клиентов |
| **Task Autopilot (lite)** | Daily Briefing + Auto Follow-up | -40% времени на рутину |

### Что НЕ в MVP (и почему):

| Отложено | Причина |
|----------|---------|
| Listing Manager | Требует интеграции с порталами — сложно, долго |
| Legal & Compliance | Юридическая верификация шаблонов — нужен юрист |
| Market Intelligence | Нужны данные по сделкам — договоры с поставщиками |
| Content Studio | Nice-to-have, не критично для PMF |
| Analytics Dashboard | Нужны данные за 2–3 месяца использования |

### Детальные User Stories (MVP)

#### Lead Engine Lite
```
US-1: Как агент, я хочу видеть все входящие запросы (WhatsApp, email, 
      Facebook) в одном месте, чтобы не терять лиды.
US-2: Как агент, я хочу, чтобы AI автоматически оценивал каждый лид 
      (hot/warm/cold), чтобы я фокусировался на горячих.
US-3: Как агент, я хочу, чтобы бот отвечал на запросы в нерабочие часы 
      и записывал клиента на звонок, чтобы не терять лиды ночью.
```

#### Client Manager Lite
```
US-4: Как агент, я хочу видеть полную историю взаимодействий с клиентом 
      (звонки, сообщения, показы) в одной карточке.
US-5: Как агент, я хочу получать алерт, когда клиент не выходит на связь 
      3+ дней, с рекомендацией действия.
US-6: Как агент, я хочу видеть всех клиентов в виде воронки 
      (новый → квалифицирован → показы → оффер → сделка).
```

#### Task Autopilot Lite
```
US-7: Как агент, я хочу каждое утро получать briefing: задачи дня, 
      горячие лиды, дедлайны.
US-8: Как агент, я хочу, чтобы follow-up после показа отправлялся 
      автоматически через 24ч/72ч/7 дней.
US-9: Как агент, я хочу, чтобы напоминания о дедлайнах (контракты, 
      регистрация, звонки) создавались автоматически из переписки.
```

### Технический MVP стек

| Компонент | Решение | Стоимость/мес |
|-----------|---------|---------------|
| Frontend | Next.js + Tailwind | $0 (Vercel free tier) |
| Backend | Supabase (auth + DB + realtime) | $25 |
| AI | Claude API / GPT-4 Mini | ~$100–200 |
| WhatsApp | Twilio WhatsApp Business API | $50 + per-message |
| Email | SendGrid | $0 (free tier) |
| Hosting | Vercel | $20 |
| **TOTAL** | | **~$200–$300/мес** |

### MVP Timeline

| Неделя | Задача | Deliverable |
|--------|--------|-------------|
| 1–2 | Supabase schema + auth + basic UI | Working login, CRM skeleton |
| 3–4 | WhatsApp integration + Multi-channel inbox | Unified inbox (WhatsApp + email) |
| 5–6 | AI Lead Scoring + Auto-response bot | Lead scoring working, bot live |
| 7–8 | Client Manager + Ghost Detector | CRM cards, funnel view, alerts |
| 9–10 | Task Autopilot: Daily Briefing + Follow-up | Morning email, auto follow-up |
